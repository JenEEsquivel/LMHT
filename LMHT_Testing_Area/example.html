<div>

English Base Word: division

Spanish Word Equivalent: división

Cómo Recordar

'div' is probablemente el más genericó elemento de HTML que hay. No tiene que tener un significado inherente o difundir cualquier estilo. Es ultilizado simplemente a "dividir" (o juntar) el contenido dentro de una pagina.

How to Remember

"div" is probably the most generic HTML element there is. It doesn't have any inherent or built-in meaning or impart any styles. It is used simply to "divide" (or group together) content within a page.


  <h1>This is a header. Esto es un encabezamiento.</h1>
  <div class="subhead">
    <p>lorem ipsum something something</p>
  </div>
